## 📝 Descrição
Este é o backend principal do sistema, desenvolvido em **Java 17** com o framework **Spring Boot**. O foco deste módulo é a implementação de um CRUD robusto para o gerenciamento de pessoas, garantindo a integridade dos dados e regras de negócio.

## 🛠️ Tecnologias e Ferramentas
* **Linguagem:** Java 17.
* **Framework:** Spring Boot.
* **IDE:** IntelliJ IDEA.
* **Persistência:** Spring Data JPA.

## 📂 Estrutura de Arquivos
* `src/main/java/.../controller`: Endpoints da API.
* `src/main/java/.../entitys`: Mapeamento das tabelas do banco.
* `src/main/java/.../repository`: Interfaces de acesso aos dados.
