## 📝 Sobre o Projeto
Esta é uma implementação de backend utilizando **NestJS** para o gerenciamento de pessoas. O objetivo é demonstrar uma arquitetura modular e escalável em **TypeScript**, desenvolvida integralmente no **IntelliJ IDEA**.

## 📂 Estrutura de Arquivos
* **src/**: Contém os módulos, controllers e services da API.
* **main.ts**: Arquivo de inicialização do servidor.
* **package.json**: Gestão de dependências do Node.js.

## 🛠️ Tecnologias
* NestJS, TypeScript e Node.js.
